## 📊 型態類型

### 反轉型態

#### 避雷針 ⚡
```
特徵：上影線 ≥ 實體 × 2
意義：頂部反轉，看跌
最佳位置：阻力位附近
```

#### 倒錘子 🔨
```
特徵：下影線 ≥ 實體 × 2
意義：底部反轉，看漲
最佳位置：支撐位附近
```

#### 頭肩頂/底 👤
```
特徵：左肩-頭部-右肩
意義：經典反轉型態
確認：跌破/突破頸線
```

### 陷阱型態

#### 假突破 🚫⬆️
```
特徵：突破 < 0.2% + 回撤 ≥ 50%
意義：誘多陷阱，看跌
勝率：⭐⭐⭐⭐⭐
```

#### 假跌破 🚫⬇️
```
特徵：跌破 < 0.2% + 反彈 ≥ 50%
意義：誘空陷阱，看漲
勝率：⭐⭐⭐⭐⭐
```

## ❓ 常見問題

### Q1: 沒有看到型態警報？

**A:** 型態不是每時每刻都出現，這是正常的。

**解決方案**：
- 查看多個交易對
- 查看多個時區
- 等待市場出現型態

### Q2: 如何提高準確率？

**A:** 使用多時區確認。

```
1h 時區：⚡ 避雷針
4h 時區：🚫⬆️ 假突破
1d 時區：⚡ 避雷針

→ 多個時區一致 = 信號更可靠
```

### Q3: 支撐阻力不準確？

**A:** 調整參數或增加回看K線數量。

```python
# 在 src/analysis/pattern_detector.py 中
self.sr_lookback = 200  # 增加到200根K線
self.sr_proximity = 0.01  # 放寬價格接近度
```

### Q4: 如何自動監控？

**A:** 使用Python腳本。

```python
import time
from src.analysis.market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer()

while True:
    alerts = analyzer.get_pattern_alerts('BTCUSDT', '1h')
    if alerts:
        for alert in alerts:
            print(f"🚨 {alert}")
    time.sleep(300)  # 每5分鐘檢查
```

