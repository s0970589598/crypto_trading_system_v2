## 故障排除

### 問題 1: 按鈕點擊無反應
**可能原因**: Streamlit 版本問題

**解決方案**:
```bash
pip install --upgrade streamlit
```

### 問題 2: 執行超時
**錯誤訊息**: `TimeoutExpired`

**解決方案**:
- 正常現象，回測需要時間
- 如果超過 5 分鐘，檢查 backtest_leverage_comparison.py 是否正常

### 問題 3: 執行失敗
**錯誤訊息**: `❌ 執行失敗`

**解決方案**:
1. 查看錯誤日誌
2. 確認 backtest_leverage_comparison.py 存在
3. 手動測試腳本：
   ```bash
   python3 backtest_leverage_comparison.py
   ```

### 問題 4: 沒有顯示結果
**可能原因**: CSV 檔案未生成

**解決方案**:
1. 檢查是否有 CSV 檔案：
   ```bash
   ls -lh leverage_comparison_*.csv
   ```
2. 如果沒有，手動執行腳本確認是否正常
