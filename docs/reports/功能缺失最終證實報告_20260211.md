## 工具 1: quantitative_risk_analysis.py

### 核心邏輯特徵 (Atomic Logic Features)

#### 1.1 Kelly Criterion 最優倉位計算
```python
# 數學公式
f* = (bp - q) / b
其中：
- f* = 最優倉位比例
- b = 賠率 (平均獲利/平均虧損)
- p = 勝率
- q = 敗率 (1 - p)

# 破產風險公式
RoR = (q/p)^(C/A)
其中：
- C = 當前資金
- A = 平均獲利金額
```

**輸入維度**:
- 勝率 (win_rate)
- 平均獲利 (avg_win)
- 平均虧損 (avg_loss)
- 初始資金 (initial_capital)

**輸出結果**:
- kelly_optimal_size: 最優倉位比例 (0-1)
- recommended_size: 建議倉位 (Half Kelly)
- kelly_ror: Kelly 破產風險 (0-1)

#### 1.2 Tilt Score 傾斜行為評分
```python
# 檢測邏輯
for i in range(len(trades) - 1):
    if current_trade.is_loss:
        leverage_increase = (next_leverage - current_leverage) / current_leverage * 100
        quantity_increase = (next_quantity - current_quantity) / current_quantity * 100
        
        if leverage_increase > 20 or quantity_increase > 20:
            # 記錄為傾斜行為
            tilt_cases.append(...)
```

**輸入維度**:
- 交易序列 (時間排序)
- 每筆交易的槓桿 (leverage)
- 每筆交易的倉位 (quantity)
- 每筆交易的盈虧 (pnl)

**輸出結果**:
- has_tilt: 是否檢測到傾斜
- severity: 嚴重程度 (none/low/medium/high)
- tilt_cases_count: 傾斜案例數量
- avg_leverage_change_after_loss: 虧損後平均槓桿變化

#### 1.3 情緒失控係數 (Emotional Control Score)
```python
# 評分邏輯 (0-100)
score = 100.0

# 扣分項目
if frequency_increase > 50:  # 虧損後下單頻率增加 > 50%
    score -= 30
if leverage_increase_pct > 30:  # 虧損後槓桿增加 > 30%
    score -= 30
if emotional_cases > 30% of trades:  # 情緒失控案例 > 30%
    score -= 20
```

**輸入維度**:
- 交易時間間隔 (time_between_trades)
- 虧損後的交易間隔 (after_loss_intervals)
- 獲利後的交易間隔 (after_win_intervals)
- 槓桿變化 (leverage_change)

**輸出結果**:
- emotional_control_score: 情緒控制評分 (0-100)
- severity: 嚴重程度 (none/low/medium/high/critical)
- frequency_increase_after_loss: 虧損後下單頻率增加百分比
- cases_count: 情緒失控案例數量

#### 1.4 能力維度評分 (Skill Dimension Scoring)
```python
# 五大維度評分 (0-10)
1. direction_judgment: 方向研判力 (基於勝率)
2. risk_management: 風險控管力 (基於最大虧損 vs 平均獲利)
3. psychological_resilience: 心理韌性 (基於傾斜行為 + 情緒控制)
4. execution_discipline: 執行紀律 (基於槓桿/持倉時間一致性)
5. cost_awareness: 成本意識 (基於手續費效率)

overall_score = (sum of 5 dimensions) / 5
```

**輸入維度**:
- 勝率 (win_rate)

- 平均獲利 (avg_win)
- 槓桿標準差 (leverage_std)
- 持倉時間標準差 (holding_time_std)
- 短線交易期望值 (short_term_expectancy)
- 手續費佔比 (fee_to_loss_ratio)

**輸出結果**:
- overall_score: 綜合評分 (0-10)
- 各維度評分 (0-10)
- deduction_reasons: 各維度扣分原因
