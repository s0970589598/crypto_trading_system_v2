## 功能詳情

### 執行按鈕
```python
if st.button("🚀 執行槓桿對比回測", use_container_width=True, type="primary"):
    with st.spinner("正在執行槓桿對比回測，請稍候..."):
        import subprocess
        result = subprocess.run(
            ['python3', 'backtest_leverage_comparison.py'],
            capture_output=True,
            text=True,
            timeout=300  # 5 分鐘超時
        )
```

### 功能特點
1. ✅ **一鍵執行** - 無需切換到終端
2. ✅ **進度提示** - 顯示 spinner 和提示訊息
3. ✅ **超時保護** - 5 分鐘超時防止卡住
4. ✅ **結果反饋** - 成功顯示 ✅ 和氣球動畫
5. ✅ **日誌查看** - 可展開查看執行日誌
6. ✅ **自動刷新** - 完成後自動重新載入頁面顯示新結果
7. ✅ **錯誤處理** - 失敗時顯示錯誤訊息

## 技術細節

### 超時處理
```python
timeout=300  # 5 分鐘超時
```
- 防止長時間卡住
- 如果超時會拋出異常

### 日誌顯示
```python
st.code(result.stdout[-2000:])  # 顯示最後 2000 字符
```
- 只顯示最後 2000 字符
- 避免界面過長

### 自動刷新
```python
st.rerun()  # 重新載入頁面
```
- 執行成功後自動刷新
- 立即顯示新結果
