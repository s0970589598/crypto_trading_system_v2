# 綜合建議邏輯改進說明

## 用戶問題

用戶看到的結果：
```
綜合建議：做多 (Long)
信心度：⭐⭐ 中

多時區趨勢：
- 1d 🔴 strong_downtrend
- 4h 🔴 strong_downtrend
- 1h 🔴 strong_downtrend
- 15m 🔴 strong_downtrend
- 5m 🟢 uptrend
- 3m 🟢 uptrend
- 1m 🟢 strong_uptrend

看多時區：3/7
看空時區：4/7
```

**疑問**：為什麼看空時區有 4 個，看多時區只有 3 個，系統還建議做多？

## 原始邏輯的問題

### 舊代碼（有問題）

```python
# 判斷方向
if bullish_count >= 3 and rsi < 70:
    direction = '做多 (Long)'
    confidence = 'high' if bullish_count == 4 else 'medium'
elif bearish_count >= 3 and rsi > 30:
    direction = '做空 (Short)'
    confidence = 'high' if bearish_count == 4 else 'medium'
else:
    direction = '觀望 (Wait)'
    confidence = 'low'
```

### 問題分析

1. **只檢查第一個條件**：使用 `if-elif` 結構，如果第一個條件符合就不會檢查第二個
2. **沒有考慮多空力量對比**：只要看多時區 >= 3 個，就建議做多，即使看空時區更多
3. **不合理的決策**：在用戶的例子中：
   - bullish_count = 3 ✅
   - rsi < 70 ✅（假設）
   - 第一個條件符合 → 建議做多
   - 但實際上 bearish_count = 4（更多！）

## 改進後的邏輯

### 新代碼（已修復）

```python
# 1. 判斷方向（改進邏輯：考慮多空力量對比）
# 計算多空力量差距
net_bullish = bullish_count - bearish_count

# 決策邏輯：
# - 如果多空力量差距明顯（>=2），跟隨主要趨勢
# - 如果多空力量接近（差距<2），結合 RSI 判斷
if net_bullish >= 2 and rsi < 70:
    # 明顯看多（多頭時區比空頭多 2 個以上）
    direction = '做多 (Long)'
    confidence = 'high' if net_bullish >= 3 else 'medium'
elif net_bullish <= -2 and rsi > 30:
    # 明顯看空（空頭時區比多頭多 2 個以上）
    direction = '做空 (Short)'
    confidence = 'high' if net_bullish <= -3 else 'medium'
elif net_bullish >= 1 and rsi < 40:
    # 輕微看多 + RSI 超賣
    direction = '做多 (Long)'
    confidence = 'medium'
elif net_bullish <= -1 and rsi > 60:
    # 輕微看空 + RSI 超買
    direction = '做空 (Short)'
    confidence = 'medium'
else:
    # 多空力量均衡或信號不明確
    direction = '觀望 (Wait)'
    confidence = 'low'
```

### 改進要點

1. **計算多空力量差距**：`net_bullish = bullish_count - bearish_count`
   - 正數：多頭優勢
   - 負數：空頭優勢
   - 零：多空均衡

2. **分級決策**：
   - **明顯優勢**（差距 >= 2）：直接跟隨主要趨勢
   - **輕微優勢**（差距 = 1）：結合 RSI 判斷
   - **均衡**（差距 = 0）：觀望

3. **信心度評估**：
   - 差距越大，信心度越高
   - 結合 RSI 的決策，信心度為中等

## 用戶案例重新分析

### 數據
- 看多時區：3 個（5m, 3m, 1m）
- 看空時區：4 個（1d, 4h, 1h, 15m）
- net_bullish = 3 - 4 = **-1**（空頭輕微優勢）

### 新邏輯判斷

1. 檢查 `net_bullish >= 2`？ → ❌（-1 < 2）
2. 檢查 `net_bullish <= -2`？ → ❌（-1 > -2）
3. 檢查 `net_bullish >= 1`？ → ❌（-1 < 1）
4. 檢查 `net_bullish <= -1 and rsi > 60`？
   - net_bullish = -1 ✅
   - 如果 rsi > 60 → **做空 (Short)**，信心度：中
   - 如果 rsi <= 60 → 繼續檢查
5. 如果都不符合 → **觀望 (Wait)**，信心度：低

### 可能的結果

根據 1h RSI 值：
- **RSI > 60**（超買）→ 做空 (Short)，信心度：⭐⭐ 中
- **30 < RSI <= 60**（中性）→ 觀望 (Wait)，信心度：⭐ 低
- **RSI < 30**（超賣）→ 觀望 (Wait)，信心度：⭐ 低
