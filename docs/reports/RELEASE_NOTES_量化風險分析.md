## 🎉 概述

我們很高興地宣布量化風險分析功能的正式發布！這是一個全面的交易風險評估工具集，提供 11 種核心分析方法，幫助交易者科學地評估和管理風險。

---

## ✨ 新功能

### 1. 核心分析功能（11種）

#### 📊 Kelly Criterion
- 計算最優倉位大小
- 提供科學的資金管理建議
- 評估破產風險

#### 🎯 傾斜行為檢測
- 識別情緒化交易
- 檢測虧損後的衝動行為
- 預防情緒失控

#### ⚠️ 破產風險分析
- 評估交易策略的長期生存能力
- 計算連續虧損概率
- 提供風險預警

#### 💰 手續費壓力分析
- 量化手續費對績效的影響
- 計算手續費/虧損比
- 優化交易頻率

#### 📈 恢復係數計算
- 評估從回撤中恢復的能力
- 衡量策略韌性
- 比較不同策略的恢復能力

#### 🧠 情緒控制分析
- 分析虧損後的行為變化
- 評估情緒控制能力
- 識別情緒失控案例

#### 🎓 能力維度評分
- 多維度評估交易技能
- 方向判斷、風險管理、心理韌性等
- 提供改進方向

#### 📉 最長連損分析
- 識別最長連續虧損期
- 評估連損期間的損失
- 提供風險預警

#### ⚡ 短線交易分析
- 評估短線交易的效果
- 計算短線交易期望值
- 提供優化建議

#### 🔄 短線交易模擬
- 模擬停止短線交易的效果
- 評估改善潛力
- 提供決策依據

#### 🛑 冷靜期建議
- 基於數據的休息建議
- 識別需要冷靜的時機
- 預防過度交易

### 2. 多種使用方式

#### Web 界面
- 交易評分頁面自動顯示所有量化指標
- 直觀的可視化展示
- 實時更新

#### CLI 工具
```bash

```
- 11 種分析類型
- 2 種輸出格式（text/json）
- 靈活的參數配置

#### 核心模組整合
- RiskManager 整合
- PerformanceMonitor 整合

- 可選啟用，不影響現有功能

#### 直接 API 調用
```python

```
