### 2. Edge Case 處理邏輯差異

#### **backtest_multi_timeframe.py** vs **cli_commands/backtest.py**

**獨立腳本的優勢**：
```python
# 獨立腳本：直接執行，無需參數解析
python3 backtest_multi_timeframe.py

# CLI 系統：需要參數
python3 cli.py backtest --strategy multi-timeframe-aggressive
```

**Edge Case**：
- 獨立腳本：硬編碼策略 ID，適合快速測試
- CLI 系統：需要指定策略 ID，更靈活但更複雜

**建議**：保留獨立腳本作為「快速測試工具」
