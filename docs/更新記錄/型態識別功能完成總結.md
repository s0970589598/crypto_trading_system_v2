## 🎯 功能特點

### 1. 智能型態識別

**8種經典型態**：
- ⚡ 避雷針（反轉向下）
- 🔨 倒錘子（反轉向上）
- 🚫⬆️ 假突破（誘多陷阱）
- 🚫⬇️ 假跌破（誘空陷阱）
- 👤⬇️ 頭肩頂（經典反轉）
- 👤⬆️ 頭肩底（經典反轉）
- ⛰️⛰️⬇️ 雙頂（看跌）
- 🏔️🏔️⬆️ 雙底（看漲）

**強度評分系統**：
- 0-100分量化信號強度
- 只顯示強度 ≥ 60 的高質量信號
- 避免噪音干擾

### 2. 支撐阻力有效性

**數學邏輯**：
```python
強度 = 觸碰次數得分(60%) + 時間因素得分(40%)
```

**評級標準**：
- 100分：觸碰10+次，非常強
- 80-99分：觸碰6-9次，強
- 60-79分：觸碰4-5次，中等
- 40-59分：觸碰2-3次，弱
- < 40分：觸碰1次，無效

**顯示信息**：
- 價格水平
- 觸碰次數
- 強度評分（視覺化進度條）
- 距離當前價格的百分比

### 3. 實時警報系統

**自動偵測**：
- 每個時區獨立分析
- 最近10根K線掃描
- 只顯示高強度信號（≥60）

**警報顯示**：
- 🚨 醒目的警報標題
- 表情符號快速識別
- 詳細描述和強度評分
- 根據型態類型使用不同顏色

## 🔧 可調整參數

在 `src/analysis/pattern_detector.py` 中：

```python
class PatternDetector:
    def __init__(self):
        # 避雷針參數
        self.lightning_rod_ratio = 2.0          # 影線/實體比例
        self.lightning_rod_body_ratio = 0.3     # 實體/總長度比例
        
        # 假突破參數
        self.fakeout_threshold = 0.002          # 突破幅度 0.2%
        self.fakeout_retracement = 0.5          # 回撤比例 50%
        self.fakeout_lookback = 20              # 回看K線數
        
        # 頭肩頂參數
        self.hs_tolerance = 0.02                # 肩膀容差 2%
        self.hs_min_bars = 15                   # 最小K線數
        
        # 支撐阻力參數
        self.sr_proximity = 0.005               # 價格接近度 0.5%
        self.sr_lookback = 100                  # 回看K線數
        self.sr_min_touches = 2                 # 最少觸碰次數
```
