## 🎯 功能詳情

### 1. K線圖主圖（40%高度）

包含：
- ✅ **K線圖**：綠色上漲 / 紅色下跌
- ✅ **EMA 12**：橙色短期均線
- ✅ **EMA 26**：藍色中期均線
- ✅ **布林帶**：上軌、中軌、下軌（灰色虛線）
- ✅ **型態標記**：自動標記避雷針、假突破等（星形標記）

### 2. MACD 指標（15%高度）

包含：
- ✅ **MACD 線**：藍色
- ✅ **Signal 線**：橙色
- ✅ **MACD 柱狀圖**：綠色（正值）/ 紅色（負值）

### 3. RSI 指標（15%高度）

包含：
- ✅ **RSI 線**：紫色
- ✅ **超買線**：70（紅色虛線）
- ✅ **超賣線**：30（綠色虛線）

### 4. ATR 波動率（15%高度）

包含：
- ✅ **ATR 線**：棕色
- ✅ **填充區域**：淺棕色

### 5. 成交量（15%高度）

包含：
- ✅ **成交量柱**：綠色上漲 / 紅色下跌
- ✅ **成交量均線**：橙色線

## 🎓 技術指標解讀

### EMA（指數移動平均線）

```
EMA 12 > EMA 26 → 上升趨勢
EMA 12 < EMA 26 → 下降趨勢
價格在 EMA 上方 → 強勢
```

### 布林帶

```
價格觸及上軌 → 可能超買
價格觸及下軌 → 可能超賣
布林帶收窄 → 可能突破
```

### MACD

```
MACD > Signal → 看多
MACD 金叉 → 買入信號
MACD 死叉 → 賣出信號
```

### RSI

```
RSI > 70 → 超買
RSI < 30 → 超賣
RSI 背離 → 趨勢反轉
```

### ATR

```
ATR 上升 → 波動增加
ATR 下降 → 波動減小
止損距離 = ATR × 1.5-2.0
```
