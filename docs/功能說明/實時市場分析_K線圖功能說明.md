# 實時市場分析 - K線圖與技術指標功能

## ✨ 新增功能

在「實時市場分析」頁面的各時區分析中，現在可以查看：

### 📊 完整的K線圖與技術指標

每個時區（1m, 3m, 5m, 15m, 1h, 4h, 1d）都包含：

#### 1. K線圖 + 均線 + 布林帶
- **K線圖**：綠色（上漲）/ 紅色（下跌）
- **EMA 12**：橙色線（短期均線）
- **EMA 26**：藍色線（中期均線）
- **布林帶**：灰色虛線（上軌、中軌、下軌）
- **型態標記**：⚡ 避雷針、🚫 假突破等（如果有）

#### 2. MACD 指標
- **MACD 線**：藍色
- **Signal 線**：橙色
- **MACD 柱狀圖**：綠色（正值）/ 紅色（負值）

#### 3. RSI 指標
- **RSI 線**：紫色
- **超買線**：70（紅色虛線）
- **超賣線**：30（綠色虛線）

#### 4. ATR 波動率
- **ATR 線**：棕色
- **填充區域**：淺棕色

#### 5. 成交量
- **成交量柱**：綠色（上漲）/ 紅色（下跌）
- **成交量均線**：橙色線

## 🔍 技術指標解讀

### EMA（指數移動平均線）

**用途**：判斷趨勢方向

```
EMA 12 > EMA 26 → 上升趨勢（做多）
EMA 12 < EMA 26 → 下降趨勢（做空）
EMA 交叉 → 趨勢轉換信號
```

### 布林帶（Bollinger Bands）

**用途**：判斷超買超賣和波動

```
價格觸及上軌 → 可能超買
價格觸及下軌 → 可能超賣
布林帶收窄 → 波動減小，可能突破
布林帶擴張 → 波動增加，趨勢加速
```

### MACD

**用途**：判斷動能和趨勢轉換

```
MACD > Signal → 看多
MACD < Signal → 看空
MACD 金叉（向上穿越 Signal）→ 買入信號
MACD 死叉（向下穿越 Signal）→ 賣出信號
柱狀圖由負轉正 → 動能轉強
```

### RSI（相對強弱指標）

**用途**：判斷超買超賣

```
RSI > 70 → 超買，可能回調
RSI < 30 → 超賣，可能反彈
RSI 50 → 中性
RSI 背離 → 趨勢可能反轉
```

### ATR（平均真實波幅）

**用途**：衡量波動率，設置止損

```
ATR 上升 → 波動增加
ATR 下降 → 波動減小
止損距離 = ATR × 1.5-2.0
```

### 成交量

**用途**：確認趨勢和信號

```
價格上漲 + 成交量放大 → 上漲確認
價格下跌 + 成交量放大 → 下跌確認
價格突破 + 成交量放大 → 突破有效
成交量萎縮 → 趨勢可能結束
```

## 🎓 多時區分析策略

### 策略1：大小週期共振

```
1. 查看 1d 時區 → 確定大趨勢
2. 查看 4h 時區 → 確定中期趨勢
3. 查看 1h 時區 → 尋找進場點
4. 查看 15m 時區 → 精確進場時機

如果所有時區趨勢一致 → 高勝率機會
```

### 策略2：背離交易

```
1. 價格創新高，但 RSI 未創新高 → 頂背離（看跌）
2. 價格創新低，但 RSI 未創新低 → 底背離（看漲）
3. 配合 MACD 確認
4. 等待 K線型態確認
```

### 策略3：突破交易

```
1. 查看布林帶收窄 → 醞釀突破
2. 查看成交量萎縮 → 等待爆發
3. 價格突破 + 成交量放大 → 進場
4. 設置止損在突破點下方
```
