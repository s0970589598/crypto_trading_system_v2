# 市場數據缺失檢測與修復

## 📋 問題說明

市場數據中可能會出現缺失的K線，原因包括：

1. **API下載中斷**：網絡問題導致部分數據未下載
2. **時間跳躍**：某些時段沒有交易數據
3. **數據更新失敗**：更新時出現錯誤

### 缺失數據的影響

- ❌ 技術指標計算不準確（EMA、MACD等）
- ❌ K線圖顯示不連續
- ❌ 型態識別可能失效
- ❌ 回測結果不可靠

### 缺失檢測算法

```python
def detect_missing_data(df, interval):
    interval_delta = get_interval_timedelta(interval)
    missing_timestamps = []
    
    for i in range(len(df) - 1):
        current_time = df.loc[i, 'timestamp']
        next_time = df.loc[i + 1, 'timestamp']
        expected_next = current_time + interval_delta
        
        if next_time != expected_next:
            # 計算缺失數量
            time_diff = next_time - current_time
            missing_count = int(time_diff / interval_delta) - 1
            
            # 記錄缺失的時間戳
            for j in range(1, missing_count + 1):
                missing_time = current_time + interval_delta * j
                missing_timestamps.append(missing_time)
    
    return missing_timestamps
```

## 🎉 總結

### 為什麼需要檢測缺失數據？

1. **確保技術分析準確**
   - EMA、MACD等指標需要連續數據
   - 缺失數據會導致指標失真

2. **K線圖完整性**
   - 缺失數據會導致圖表不連續
   - 影響視覺判斷

3. **型態識別可靠性**
   - 型態識別需要完整的K線序列
   - 缺失數據可能導致誤判

4. **回測準確性**
   - 回測需要完整的歷史數據
   - 缺失數據會影響回測結果
