# CMT Level 3 專業分析功能說明

## 功能概述

根據用戶要求，添加了機構級技術圖表分析師（CMT Level 3）的專業分析功能，回答交易員必須回答的關鍵問題。

## 新增分析內容

### 1. 📊 1H/4H 大級別趨勢

**分析內容**：
- 趨勢判斷：強勢上漲/上漲趨勢/強勢下跌/下跌趨勢/區間震盪
- K線排列描述：EMA 多頭/空頭排列，價格創新高/新低

**判斷邏輯**：
```python
if 'strong_uptrend' in h4_trend or 'strong_uptrend' in h1_trend:
    major_trend = "強勢上漲"
    major_trend_desc = "EMA 多頭排列，價格持續創新高"
elif 'uptrend' in h4_trend or 'uptrend' in h1_trend:
    major_trend = "上漲趨勢"
    major_trend_desc = "價格在主要均線上方運行"
# ... 其他情況
```

**顯示效果**：
```
📊 1H/4H 大級別趨勢
┌─────────────────────────────────────┐
│ 趨勢判斷：強勢下跌                    │
│ EMA 空頭排列，價格持續創新低          │
└─────────────────────────────────────┘
```

### 2. 🎯 開倉理由評估

**分析內容**：
- 進場類型：順勢回調/逆勢摸頂/逆勢摸底/區間交易/等待信號
- 有效性評估：有效/高風險/觀望

**判斷邏輯**：
- **順勢回調**：大級別上漲 + 小級別（15m/5m）回調後做多
- **逆勢摸頂/底**：大級別上漲時做空（或下跌時做多）→ 高風險警告
- **等待信號**：小級別未出現明確方向

**顯示效果**：
```
🎯 開倉理由評估
進場類型：逆勢摸底
┌─────────────────────────────────────┐
│ 評估：高風險                          │
│ ⚠️ 大級別下跌時做多，風險較高         │
└─────────────────────────────────────┘
```

### 3. 📈 15m/5m 入場級別信號

**分析內容**：
- K線反轉信號識別
- RSI 超買超賣
- MACD 金叉死叉

**判斷邏輯**：
```python
reversal_signals = []

if rsi > 70:
    reversal_signals.append("RSI 超買 (>70)")
elif rsi < 30:
    reversal_signals.append("RSI 超賣 (<30)")

if 'golden_cross' in macd_state:
    reversal_signals.append("MACD 金叉（看多信號）")
elif 'death_cross' in macd_state:
    reversal_signals.append("MACD 死叉（看空信號）")
```

**顯示效果**：
```
📈 15m/5m 入場級別信號
✅ 發現反轉信號：RSI 超賣 (<30)、MACD 金叉（看多信號）

或

❌ 無明確反轉信號，建議空倉等待
```

### 4. 🏗️ 市場結構分析

**分析內容**：
- 結構類型：Higher Highs (多頭結構)/Lower Lows (空頭結構)/區間震盪 (Range)
- 結構描述：價格創新高/新低的模式

**顯示效果**：
```
🏗️ 市場結構分析
┌──────────────────────┬──────────────────────┐
│ 結構類型              │ 結構描述              │
│ Lower Lows (空頭結構) │ 價格持續創新低，      │
│                      │ 反彈不破前高          │
└──────────────────────┴──────────────────────┘
```

### 5. 📉 技術指標分析

**分析內容**：
- **布林帶位置**：價格突破上軌/跌破下軌/在上半部/在下半部/在中軌附近
- **動能分析**：動能增強（看多）/動能減弱（看空）/動能中性
- **關鍵指標**：RSI 值、MACD 狀態

**顯示效果**：
```
📉 技術指標分析
┌─────────────┬─────────────┬─────────────┐
│ 布林帶位置   │ 動能分析     │ 關鍵指標     │
│ 價格跌破下軌 │ 動能減弱     │ RSI: 28.5   │
│ 可能超賣     │ （看空）     │ MACD: 死叉  │
│ 💡 警惕反彈  │             │             │
└─────────────┴─────────────┴─────────────┘
```

### 6. ✅ 綜合評估

**評估結果**：
- ✅ 符合進場條件（有效 + 有信號）
- ⚠️ 逆勢交易，風險極高，建議空倉
- ❌ 無明確信號，建議空倉等待
- ⏸️ 觀望，等待更好機會

**判斷邏輯**：
```python
if entry_validity == "有效" and has_reversal:
    overall_assessment = "✅ 符合進場條件"
elif entry_validity == "高風險":
    overall_assessment = "⚠️ 逆勢交易，風險極高，建議空倉"
elif not has_reversal:
    overall_assessment = "❌ 無明確信號，建議空倉等待"
else:
    overall_assessment = "⏸️ 觀望，等待更好機會"
```

### 7. 📋 專業交易計劃

**包含內容**：

#### 進場區間
- 當前價格
- 建議進場價位（現價/等待回調）

#### 止損價格（必須無條件執行）
- 止損位（具體價格）
- 止損方法（支撐位下方/阻力位上方/1.5 ATR）
- 止損距離（金額和百分比）

#### 止盈價格
- 第一止盈位（平倉 50%）
- 第二止盈位（推保護止損）
- 止盈方法（阻力位前/支撐位前/3 ATR）

#### 盈虧比 (R/R)
- 盈虧比數值
- 評估：優秀 (>2.0)/良好 (>1.5)/一般 (<1.5)

#### 關鍵價位
- 支撐位（價格 + 距離百分比）
- 阻力位（價格 + 距離百分比）

#### 建議倉位
- 基於 2% 風險管理原則
- 計算公式：(總資金 × 2%) ÷ 止損金額
- 實例計算：10,000 USDT 本金的建議倉位

**顯示效果**：
```
📋 專業交易計劃

進場區間：
- 當前價格：$69,102.90
- 建議進場：等待回調至 $68,500 (支撐位附近)

止損價格 (觸發後必須無條件執行)：
- 止損位：$68,200 (支撐位下方)
- 止損距離：902.90 (1.31%)

止盈價格：
- 第一止盈位（平倉 50%）：$70,500 (阻力位前)
- 第二止盈位（推保護止損）：建議在第一止盈後，將止損移至成本價

盈虧比 (R/R)：
- 盈虧比：1.55
- 評估：✅ 良好 (>1.5)

關鍵價位：
- 支撐位：$68,350 (距離 1.09%)
- 阻力位：$70,800 (距離 2.46%)

建議倉位：
- 基於 2% 風險管理原則
- 單筆最大虧損 = 總資金 × 2%
- 建議倉位 = (總資金 × 2%) ÷ 止損金額
- 例如：10,000 USDT 本金 → 最大虧損 200 USDT → 倉位 = 200 ÷ 902.90 = 0.2215 單位
```

## 回答的關鍵問題

根據用戶要求，現在系統可以回答：

1. ✅ **1H/4H 大級別趨勢**：上漲/下跌/震盪？K線排列如何？
2. ✅ **開倉理由**：順勢回調還是逆勢摸頂/底？
3. ✅ **支撐/壓力位**：在哪裡？價格是否已抵達？
4. ✅ **K線反轉信號**：15m/5m 是否有明確信號？
5. ✅ **止損位**：在哪個價格？為什麼設這裡？
6. ✅ **止盈位**：在哪裡？
7. ✅ **盈虧比**：R/R 是多少？是否 >1.5？
8. ✅ **進場區間**：具體價格範圍
9. ✅ **建議倉位**：基於 2% 風險管理
10. ✅ **市場結構**：Higher Highs/Lower Lows/Range？
11. ✅ **布林帶**：價格位置？開口/縮口？
12. ✅ **MACD/RSI**：背離？動能增強/減弱？
