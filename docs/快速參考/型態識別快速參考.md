## 💡 實戰技巧

### 避雷針 ⚡
```
✅ 出現在阻力位 + 放量 → 強烈看跌
⚠️ 出現在上升趨勢中段 → 可能只是洗盤
```

### 假突破 🚫⬆️
```
✅ 突破後立即反向 → 最佳進場點
⚠️ 需要配合成交量確認
```

### 頭肩頂 👤⬇️
```
✅ 右肩形成 + 跌破頸線 → 確認反轉
⚠️ 等待右肩完成，不要提前進場
```

### 支撐阻力 🎯
```
✅ 強度 > 80 + 多次觸碰 → 高勝率
⚠️ 突破後可能變成支撐/阻力互換
```

## ⚙️ API 使用

### 基本用法

```python
from src.analysis.market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer()

# 偵測型態
result = analyzer.detect_patterns('BTCUSDT', '1h')
for pattern in result['patterns']:
    print(f"{pattern.emoji} {pattern.description}")

# 獲取警報
alerts = analyzer.get_pattern_alerts('BTCUSDT', '1h')
for alert in alerts:
    print(alert)

# 支撐阻力分析
sr = analyzer.analyze_support_resistance_strength('BTCUSDT', '1h')
print(f"最近支撐：${sr['nearest_support']['level']:.2f}")
```
