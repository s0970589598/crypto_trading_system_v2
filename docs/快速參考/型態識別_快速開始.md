## 🎯 常見問題

### Q1: 沒有看到型態警報？

**A:** 這是正常的！型態不是每時每刻都出現。

解決方案：
- 查看多個交易對（BTCUSDT, ETHUSDT, SOLUSDT）
- 查看多個時區（1h, 4h, 1d）
- 等待市場出現型態

### Q2: 如何提高偵測靈敏度？

**A:** 修改 `src/analysis/pattern_detector.py` 的參數：

```python
# 降低避雷針的要求（更容易偵測）
self.lightning_rod_ratio = 1.5  # 原本 2.0

# 降低假突破的要求
self.fakeout_threshold = 0.003  # 原本 0.002
```

### Q3: 支撐阻力不準確？

**A:** 調整參數：

```python
# 增加回看K線數量（更多歷史數據）
self.sr_lookback = 200  # 原本 100

# 調整價格接近度
self.sr_proximity = 0.01  # 原本 0.005（更寬鬆）
```

## 💡 實戰技巧（1分鐘學會）

### 技巧1：多時區確認

```
如果 1h、4h、1d 都出現同樣的型態 → 信號更可靠
```

### 技巧2：配合支撐阻力

```
避雷針 + 強阻力（強度 > 80） → 強烈看跌
假跌破 + 強支撐（強度 > 80） → 強烈看漲
```

### 技巧3：只交易高強度信號

```
強度 ≥ 80 → 可以考慮交易
強度 60-79 → 謹慎觀察
強度 < 60 → 忽略（系統已自動過濾）
```

## 🎓 5分鐘進階教程

### 使用Python API

```python
from src.analysis.market_analyzer import MarketAnalyzer

# 創建分析器
analyzer = MarketAnalyzer()

# 偵測型態
result = analyzer.detect_patterns('BTCUSDT', '1h')

# 顯示結果
for pattern in result['patterns']:
    print(f"{pattern.emoji} {pattern.description}")
    print(f"強度：{pattern.strength:.0f}/100")

# 支撐阻力分析
sr = analyzer.analyze_support_resistance_strength('BTCUSDT', '1h')
print(f"\n最近支撐：${sr['nearest_support']['level']:.2f}")
print(f"強度：{sr['nearest_support']['strength']:.0f}/100")
print(f"觸碰：{sr['nearest_support']['touches']} 次")
```

### 自動監控腳本

```python
import time
from src.analysis.market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer()
symbols = ['BTCUSDT', 'ETHUSDT', 'SOLUSDT']

while True:
    for symbol in symbols:
        alerts = analyzer.get_pattern_alerts(symbol, '1h')
        if alerts:
            print(f"\n🚨 {symbol} 警報：")
            for alert in alerts:
                print(f"  {alert}")
    
    time.sleep(300)  # 每5分鐘檢查一次
```
