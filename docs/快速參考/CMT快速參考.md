## 🎯 核心判斷邏輯

### 大級別趨勢
- **強勢上漲**：4h/1h 有 strong_uptrend
- **上漲趨勢**：4h/1h 有 uptrend
- **強勢下跌**：4h/1h 有 strong_downtrend
- **下跌趨勢**：4h/1h 有 downtrend
- **區間震盪**：其他情況

### 開倉理由
- **順勢回調**：大級別上漲 + 小級別回調 → ✅ 有效
- **逆勢摸頂**：大級別上漲 + 小級別做空 → ⚠️ 高風險
- **逆勢摸底**：大級別下跌 + 小級別做多 → ⚠️ 高風險
- **區間交易**：大級別震盪 → 中性
- **等待信號**：小級別無明確方向 → 觀望

### 反轉信號
- **RSI 超買**：RSI > 70 → 看空信號
- **RSI 超賣**：RSI < 30 → 看多信號
- **MACD 金叉**：golden_cross → 看多信號
- **MACD 死叉**：death_cross → 看空信號
- **無信號**：以上都沒有 → ❌ 建議空倉等待

### 綜合評估
- **✅ 符合進場條件**：有效 + 有信號
- **⚠️ 高風險警告**：逆勢交易
- **❌ 建議空倉等待**：無明確信號
- **⏸️ 觀望等待**：其他情況

### 盈虧比評估
- **✅ 優秀**：R/R > 2.0
- **✅ 良好**：R/R > 1.5
- **⚠️ 一般**：R/R < 1.5

### 系統分析
```

多空力量：空頭優勢 -1
綜合建議：
  - RSI > 60 → 做空（順勢）
  - RSI < 60 → 觀望（等待信號）
```

## 🧪 測試場景

### 場景 1：順勢做多（應該通過）
```
大級別：上漲
小級別：回調
信號：RSI < 30 或 MACD 金叉
預期：✅ 符合進場條件
```

### 場景 2：逆勢做多（應該警告）
```
大級別：下跌
小級別：反彈
信號：任何
預期：⚠️ 逆勢交易，風險極高
```

### 場景 3：無信號（應該等待）
```
大級別：任何
小級別：任何
信號：無
預期：❌ 無明確信號，建議空倉等待
```
