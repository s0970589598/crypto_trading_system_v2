## 🎯 使用場景

### 場景1：我想識別K線型態
1. 閱讀：[型態識別_快速開始](快速參考/型態識別_快速開始.md)
2. 運行：`python3 測試型態識別.py`
3. 查看：[型態識別視覺化說明](功能說明/型態識別視覺化說明.md)

### 場景2：我想查看K線圖
1. 閱讀：[K線圖功能說明](功能說明/實時市場分析_K線圖功能說明.md)
2. 啟動：`./啟動Web界面v2.sh`
3. 進入：實時市場分析 → 展開時區

### 場景3：我的數據有缺失
1. 閱讀：[數據缺失檢測與修復說明](功能說明/數據缺失檢測與修復說明.md)
2. 檢測：`python3 檢測並填補缺失數據.py`
3. 修復：`python3 檢測並填補缺失數據.py --fetch`

### 場景4：我想做專業分析
1. 閱讀：[CMT_Level3專業分析功能說明](功能說明/CMT_Level3專業分析功能說明.md)
2. 啟動：`./啟動Web界面v2.sh`
3. 進入：實時市場分析 → 查看綜合建議

## 📞 快速命令

```bash
# 測試型態識別
python3 測試型態識別.py

# 檢測數據缺失
python3 檢測並填補缺失數據.py

# 修復數據缺失
python3 檢測並填補缺失數據.py --fetch

# 啟動Web界面
./啟動Web界面v2.sh

# 更新市場數據
python3 快速重新下載_關鍵時區.py
```
