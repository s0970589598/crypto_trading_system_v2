# 自動評分註記說明

## 📝 註記結構

自動評分生成的註記包含以下部分：

### 1. 【交易概況】
```
持倉 X.X 小時，盈虧 X.XX%，槓桿 Xx
```
- 基本交易信息
- 來源：交易記錄本身

---

### 2. 【市場數據】(1h 時區)
```
• 價格：2110.42
• 開盤：2105.30
• 最高：2115.67
• 最低：2098.45
```
- **時區**：1 小時（1h）
- **時間點**：交易進場時刻
- **來源**：Binance 歷史數據

---

### 3. 【技術指標】(1h 時區)
```
• EMA7: 2110.42
• EMA20: 2136.98
• EMA50: 2196.31
• 均線排列: bearish
• RSI: 33.5 (weak)
• MACD: -31.30 / Signal: -33.55 (bullish)
• ATR: 47.47 (2.26%, normal)
• 布林帶位置: lower_half
• 成交量: 42052.91 (比率: 0.66x, low)
```

#### 指標說明

**EMA（指數移動平均）**
- EMA7：7 週期短期均線
- EMA20：20 週期中期均線
- EMA50：50 週期長期均線
- 與你的交易策略一致

**均線排列**
- `bullish`：多頭排列（EMA7 > EMA20 > EMA50）
- `bearish`：空頭排列（EMA7 < EMA20 < EMA50）
- `mixed`：混合排列

**RSI（相對強弱指標）**
- 數值：0-100
- 狀態：
  - `oversold`：超賣（< 30）
  - `weak`：偏弱（30-40）
  - `neutral`：中性（40-60）
  - `strong`：偏強（60-70）
  - `overbought`：超買（> 70）

**MACD（指數平滑異同移動平均線）**
- 狀態：
  - `golden_cross`：金叉（看多信號）
  - `death_cross`：死叉（看空信號）
  - `bullish`：多頭（MACD > Signal）
  - `bearish`：空頭（MACD < Signal）

**ATR（平均真實波幅）**
- 絕對值 + 百分比
- 波動率狀態：
  - `low`：低波動（< 1.5%）
  - `normal`：正常波動（1.5-3%）
  - `high`：高波動（3-5%）
  - `very_high`：極高波動（> 5%）

**布林帶位置**
- `below_lower`：價格在下軌之下
- `lower_half`：價格在下軌與中軌之間
- `upper_half`：價格在中軌與上軌之間
- `above_upper`：價格在上軌之上

**成交量**
- 絕對值 + 與 20 日均量的比率
- 狀態：
  - `low`：低量（< 0.8x）
  - `normal`：正常（0.8-1.5x）
  - `high`：放量（1.5-2x）
  - `very_high`：爆量（> 2x）

- **時區**：1 小時（1h）
- **時間點**：交易進場時刻
- **來源**：根據市場數據計算

---

### 4. 【市場分析】(1h 時區)
```
• 趨勢：downtrend（強度：35.2/100）
• 均線排列：bearish
• 支撐位：2050.00 (距離：2.85%)
• 阻力位：2150.00 (距離：1.88%)
```

**趨勢類型**
- `strong_uptrend`：強勢上升
- `uptrend`：上升
- `sideways`：震盪
- `downtrend`：下降
- `strong_downtrend`：強勢下降

**趨勢強度**
- 0-100 分
- 基於 EMA 斜率計算
- 50 = 中性

**支撐/阻力位**
- 基於最近 50 根 K 線的高低點
- 距離：當前價格與支撐/阻力的百分比距離

- **時區**：1 小時（1h）
- **時間點**：交易進場時刻

---

### 5. 【多時區分析】
```
• 15m: downtrend, RSI=28.3, MACD=bearish
• 1h: downtrend, RSI=33.5, MACD=bullish
• 4h: sideways, RSI=45.2, MACD=bearish
• 1d: uptrend, RSI=52.1, MACD=golden_cross
```

- **包含所有時區**：15m、1h、4h、1d
- **顯示關鍵指標**：趨勢、RSI、MACD 狀態
- **用途**：對比不同時區的市場狀態

**為什麼不同時區數據不同？**
- 15m：最靈敏，適合超短線
- 1h：平衡，適合短線
- 4h：穩定，適合中線
- 1d：最穩定，適合長線

**實際應用**
- 如果所有時區趨勢一致 → 趨勢強勁
- 如果時區趨勢分歧 → 可能轉折點
- 長期時區 > 短期時區 → 參考價值更高

---

### 6. 【發現問題】
```
1. 逆勢做多（市場 downtrend）
2. RSI 超買時做多（RSI=75.2）
3. 槓桿偏高（>20x）
```

- 列出評分時發現的問題
- 影響各項評分
- 用於改進交易

---

## 🎯 為什麼主要使用 1h 時區？

1. **與策略一致**：你的多時框策略主要看 1h 和 4h
2. **最常用**：1h 是最常見的交易時區
3. **平衡性好**：不會太靈敏也不會太遲鈍
4. **數據完整**：Binance 1h 數據最完整

## 📊 如何查看其他時區？

在 Web 界面的「市場分析」標籤中：
1. 點擊「🔍 查看」按鈕
2. 切換到「🔬 市場分析」標籤
3. 使用時區選擇器切換（15m、1h、4h、1d）
4. 查看該時區的完整 K 線圖和技術指標

## 🔄 數據更新

- **自動下載**：如果市場數據不存在，會自動從 Binance 下載
- **自動更新**：如果數據超過 24 小時，會自動更新
- **歷史數據**：可以分析任何歷史時間點的市場狀態

## 💡 使用建議

1. **主要看 1h 數據**：快速了解進場時的市場狀態
2. **對比多時區**：看趨勢是否一致
3. **結合圖表**：在市場分析標籤查看完整圖表
4. **關注問題列表**：重點改進發現的問題

## ⚠️ 注意事項

- 所有數據都是**進場時刻**的市場狀態，不是現在
- 使用 **EMA** 而非 SMA，與策略一致
- 如果某個幣種 Binance 沒有數據，會跳過市場分析
- 市場分析需要網絡連接（下載數據）
