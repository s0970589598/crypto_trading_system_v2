## 🔧 開發指南

### 新增功能到交易覆盤

#### 步驟 1：創建新模組文件
```python
# pages/review/new_feature.py

"""
New Feature Module
"""

import streamlit as st
import pandas as pd
import json
from pathlib import Path
from datetime import datetime
import sys
import traceback

# 添加當前目錄到路徑
sys.path.insert(0, ".")

def render():
    """渲染新功能頁面"""
    st.subheader("🆕 新功能")
    
    # 你的代碼
    st.write("這是新功能")
```

#### 步驟 2：更新 `__init__.py`
```python
# pages/review/__init__.py

from . import bingx_analysis
from . import record_management
from . import quality_scoring
from . import loss_review
from . import new_feature  # 新增

__all__ = [
    'bingx_analysis',
    'record_management',
    'quality_scoring',
    'loss_review',
    'new_feature'  # 新增
]
```

#### 步驟 3：更新主文件
```python
# web_dashboard_v2.py

elif category == "6️⃣ 交易覆盤":
    st.header("📝 交易覆盤")
    
    from pages.review import (
        bingx_analysis, 
        record_management, 
        quality_scoring, 
        loss_review,
        new_feature  # 新增
    )
    
    sub_function = st.sidebar.selectbox(
        "選擇功能",
        [
            "BingX 交易分析",
            "交易記錄管理",
            "執行質量評分",
            "虧損分析",
            "新功能"  # 新增
        ]
    )
    
    if sub_function == "BingX 交易分析":
        bingx_analysis.render()
    elif sub_function == "交易記錄管理":
        record_management.render()
    elif sub_function == "執行質量評分":
        quality_scoring.render()
    elif sub_function == "虧損分析":
        loss_review.render()
    elif sub_function == "新功能":  # 新增
        new_feature.render()
```

## ⚠️ 注意事項

### 1. 導入順序
模組文件中的導入順序很重要：
```python
import streamlit as st  # 先導入 streamlit
import pandas as pd
# ... 其他標準庫
from pathlib import Path
# ... 最後導入自定義模組
```

### 2. 路徑問題
所有模組都添加了：
```python
import sys
sys.path.insert(0, ".")
```
確保可以正確導入其他模組。

### 3. 狀態管理
Streamlit 的 `st.session_state` 在模組間共享，可以用來傳遞數據。

### 4. 備份文件
原始文件已備份為：
```
web_dashboard_v2.py.backup_before_modularization
```

如果需要恢復：
```bash
cp web_dashboard_v2.py.backup_before_modularization web_dashboard_v2.py
```
