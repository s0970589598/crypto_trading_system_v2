# 評分註記示例

## 📋 完整註記格式

### 示例 1：順勢交易（高分）

```
【交易概況】持倉2.5小時，盈虧2.35%，槓桿10x。

【市場數據】
• 價格：67850.50
• 開盤：67200.00
• 最高：68100.00
• 最低：67150.00

【技術指標】
• SMA7: 67500.25
• SMA25: 66800.50
• SMA99: 65200.00
• RSI: 58.3 (neutral)
• MACD: 125.50 / Signal: 98.20 (bullish)
• ATR: 850.50 (1.25%, normal)
• 布林帶位置: upper_half
• 成交量: 1250.50 (比率: 1.35x, normal)

【市場分析】
• 趨勢：uptrend（強度：65.5/100）
• 均線排列：bullish
• 支撐位：66500.00 (距離：1.99%)
• 阻力位：68500.00 (距離：0.96%)

【評估結果】執行良好，未發現明顯問題。
```

**評分**：
- 進場質量：105/100（順勢+5分）
- 出場質量：100/100
- 風險管理：100/100
- 紀律遵守：100/100
- **總分：101.25/100**

## 📊 註記內容說明

### 【交易概況】
- 持倉時間
- 盈虧百分比
- 使用槓桿

### 【市場數據】
- 當前價格
- 開盤價、最高價、最低價
- 反映市場波動情況

### 【技術指標】
- **移動平均線**：SMA7/25/99，判斷趨勢
- **RSI**：相對強弱指標，判斷超買超賣
- **MACD**：趨勢動量指標，判斷買賣信號
- **ATR**：波動率指標，評估風險
- **布林帶**：價格通道，判斷超買超賣
- **成交量**：市場活躍度

### 【市場分析】
- **趨勢**：市場方向和強度
- **均線排列**：多頭/空頭/混合
- **支撐阻力位**：關鍵價格位置

### 【發現問題】或【評估結果】
- 列出所有識別的問題
- 或確認執行良好

## 💡 如何使用註記

### 1. 學習最佳實踐
- 對比高分和低分交易
- 分析成功交易的市場環境
- 學習在什麼市場狀態下交易

### 2. 識別錯誤模式
- 查看重複出現的問題
- 分析虧損交易的共同點
- 制定改進計劃

### 3. 優化進場時機
- 檢查趨勢方向是否一致
- 確認 RSI 不在極端區域
- 等待 MACD 信號確認

### 4. 改善風險管理
- 根據波動率調整槓桿
- 在高波動市場降低倉位
- 設置合理的止損位

### 5. 追蹤改善進度
- 定期查看評分趨勢
- 對比不同時期的註記
- 驗證改進措施的效果
