## 📊 三大情境使用情況

### 情境 1：實盤分析 🔴 - 需要最新數據

#### ✅ Web 界面
**文件**：`pages/trading/live_market_analysis.py`

**使用方式**：
```python
from src.analysis.market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer()

# 自動更新（載入時檢查過期）
df = analyzer.load_market_data(symbol, timeframe)

# 手動強制更新（點擊按鈕）
new_data = analyzer._fetch_binance_klines(symbol, interval, start, end)
```

**數據更新邏輯**：
- `load_market_data()` 會自動檢查數據是否過期（24小時）
- 如果過期，自動調用 `_update_market_data()` 更新
- 用戶也可以點擊「🔄 更新數據」按鈕強制更新

#### ✅ CLI 工具
**文件**：`快速更新市場數據.py`

**使用方式**：
```python
analyzer = MarketAnalyzer()

# 更新現有數據到最新
df = analyzer.load_market_data(symbol, interval)
# load_market_data 內部會自動調用 _update_market_data()
```

**結論**：✅ 已統一使用 `MarketAnalyzer`

---

### 情境 2：交易覆盤 📅 - 需要歷史特定時間數據

#### ✅ Web 界面
**文件**：`pages/review/quality_scoring.py`

**使用方式**：
```python
from src.analysis.market_analyzer import MarketAnalyzer

market_analyzer = MarketAnalyzer()

# 分析特定時間點（自動補齊缺失數據）
market_analysis = market_analyzer.analyze_market_at_time(
    symbol=symbol,
    timestamp=open_time,
    intervals=['1m', '3m', '5m', '15m', '1h', '4h', '1d']
)
```

**數據更新邏輯**：
- `analyze_market_at_time()` 會檢查時間點是否在數據範圍內
- 如果不在，自動調用 `_update_market_data_for_timestamp()` 補齊
- 用戶無感知，自動完成

#### ✅ CLI 工具
**文件**：`檢測並填補缺失數據.py`（已整合）

**使用方式**：
```python
from src.analysis.market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer()

# 檢測缺失
gaps = analyzer.detect_missing_gaps(symbol, interval)

# 填補缺失
result = analyzer.fill_missing_data(symbol, interval)
```

**結論**：✅ 已統一使用 `MarketAnalyzer`（第一步已完成）
