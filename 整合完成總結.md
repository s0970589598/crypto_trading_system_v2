## 使用指南

### 方法 1: CLI（推薦用於自動化）

#### 單策略回測
```bash
python3 cli.py backtest --strategy multi-timeframe-aggressive
```

#### 多策略回測
```bash
python3 cli.py backtest \
  --strategy multi-timeframe-aggressive \
  --strategy breakout-strategy
```

#### 槓桿對比
```bash

```
