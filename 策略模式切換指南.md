# 🎯 策略模式切換指南

## 📊 兩種策略模式對比

| 指標 | 激進模式（推薦）⭐ | 輕鬆模式 |
|------|------------------|---------|
| **止損** | **1.5 ATR** | **2.0 ATR** |
| **目標** | **3.0 ATR** | **4.0 ATR** |
| **交易次數** | 33 筆（89 天） | 22 筆（89 天） |
| **勝率** | **54.55%** ✅ | 45.45% |
| **獲利因子** | **1.86** ✅ | 1.45 |
| **總收益** | **+40.42%** ✅ | +18.82% |
| **最大回撤** | **6.68%** ✅ | 11.24% |
| **單筆風險** | ~1.5% | ~2% |
| **適合** | 想要高收益 | 想要輕鬆交易 |

---

## 🚀 如何切換策略模式

### 方法 1：修改 .env 文件（推薦）

編輯 `.env` 文件：

```bash
# 激進模式（推薦）
STRATEGY_MODE=aggressive

# 或輕鬆模式
STRATEGY_MODE=relaxed
```

然後重啟程式：

```bash
python3 trading_alert_system.py
```

---

### 方法 2：啟動時選擇

程式會自動詢問你要使用哪個模式（如果 .env 沒有設置）。

---

### 方法 3：自定義 ATR 倍數

如果你想要自定義止損和目標，可以在 `.env` 中設置：

```bash
STRATEGY_MODE=custom
STOP_LOSS_ATR=1.8
TAKE_PROFIT_ATR=3.6
```

## 💡 使用建議

### 階段 1：新手期（1-3 個月）

**推薦：輕鬆模式**

```bash
STRATEGY_MODE=relaxed
```

**原因**：
- ✅ 交易次數少（22 筆 vs 33 筆）
- ✅ 壓力小，容易堅持
- ✅ 建立信心和紀律
- ✅ 熟悉策略邏輯

---

### 階段 2：成長期（3-6 個月）

**推薦：激進模式**

```bash
STRATEGY_MODE=aggressive
```

**原因**：
- ✅ 收益更高（+40% vs +19%）
- ✅ 勝率更高（54.55% vs 45.45%）
- ✅ 回撤更小（6.68% vs 11.24%）
- ✅ 已經建立紀律，可以處理更多交易

---

### 階段 3：成熟期（6 個月+）

**推薦：根據市場調整**

**市場波動大時**（如重大新聞、暴漲暴跌）：
```bash
STRATEGY_MODE=relaxed  # 避免被掃損
```

**市場穩定時**（如橫盤整理、溫和趨勢）：
```bash
STRATEGY_MODE=aggressive  # 提高收益
```

## 🔍 如何判斷該用哪個模式？

### 用激進模式，如果你：

- ✅ 有時間盯盤（每週 2-3 筆交易）
- ✅ 紀律好，能嚴格執行止損
- ✅ 想要更高收益
- ✅ 能接受更頻繁的交易

### 用輕鬆模式，如果你：

- ✅ 時間有限（每週 1-2 筆交易）
- ✅ 剛開始學習
- ✅ 想要輕鬆的交易節奏
- ✅ 擔心「止損太緊」的心理壓力

---

## ⚠️ 重要提醒

### 不要頻繁切換！

**錯誤做法**：
- ❌ 今天用激進，明天用輕鬆
- ❌ 虧損後立即切換
- ❌ 看心情切換

**正確做法**：
- ✅ 至少使用 1 個月再評估
- ✅ 記錄每筆交易
- ✅ 統計勝率和收益
- ✅ 根據數據決定是否切換

## 🎯 常見問題

### Q1: 我應該用哪個模式？

**A**: 新手建議從輕鬆模式開始，1-3 個月後切換到激進模式。

---

### Q2: 可以同時運行兩個模式嗎？

**A**: 不建議。會收到重複通知，而且難以管理。選擇一個堅持使用。

---

### Q3: 激進模式的交易次數太多怎麼辦？

**A**: 可以切換到輕鬆模式，或者調整進場條件（如提高 RSI 範圍）。

---

### Q4: 輕鬆模式的收益太低怎麼辦？

**A**: 如果你已經建立紀律，可以切換到激進模式。或者增加倉位（但不建議超過 20%）。

---

### Q5: 可以自定義 ATR 倍數嗎？

**A**: 可以！在 .env 中設置：
```bash
STRATEGY_MODE=custom
STOP_LOSS_ATR=1.8
TAKE_PROFIT_ATR=3.6
```

但建議先使用預設的兩種模式，有經驗後再自定義。
