# K線標記位置說明

## 標記位置設計

### 做多交易（Long）
```
        ↑ 價格
        
    [K線上方空白區域]
    
    ┌─────┐
    │     │  ← K線
    │     │
    └─────┘
    
    🟢 ▲   ← 進場標記（在K線下方）
    進場
    08:50:22
```

### 做空交易（Short）
```
        ↑ 價格
        
    🔴 ▼   ← 進場標記（在K線上方）
    進場
    08:50:22
    
    ┌─────┐
    │     │  ← K線
    │     │
    └─────┘
    
    [K線下方空白區域]
```

### 獲利出場
```
        ↑ 價格
        
    🟡 ✕   ← 出場標記（在K線上方）
    出場
    09:23:54
    
    ┌─────┐
    │     │  ← K線
    │     │
    └─────┘
```

### 虧損出場
```
        ↑ 價格
        
    ┌─────┐
    │     │  ← K線
    │     │
    └─────┘
    
    🔴 ✕   ← 出場標記（在K線下方）
    出場
    09:23:54
```

## K線時間對應關係

### 15分鐘K線示例
```
時間軸：  8:30    8:45    9:00    9:15    9:30
         │       │       │       │       │
K線：    [8:30]  [8:45]  [9:00]  [9:15]  [9:30]
         
交易時間：        8:50:22         9:23:54
                    ↓               ↓
標記位置：        [8:45]          [9:15]
```

**說明**：
- 8:50:22 的交易 → 標記在 8:45 的K線（因為在 8:45-9:00 時間段內）
- 9:23:54 的交易 → 標記在 9:15 的K線（因為在 9:15-9:30 時間段內）
