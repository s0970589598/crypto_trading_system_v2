## 💡 使用建議

### 第一次使用

1. 先讀 `README.md` 了解項目
2. 再讀 `快速開始.md` 學習操作
3. 然後讀 `我的交易策略.md` 理解策略
4. 最後讀 `最終方案總結.md` 選擇配置

### 日常使用

1. 每天運行 `trading_alert_system.py`
2. 收到通知後檢查條件
3. 符合條件就交易
4. 記錄每筆交易

### 定期覆盤

1. 每週統計勝率和收益
2. 閱讀 `虧損分析與優化方案.md`
3. 分析自己的錯誤
4. 調整參數（如需要）

### 驗證策略

1. 運行 `improved_backtest.py`
2. 運行 `final_optimized_backtest.py`
3. 運行 `compare_stop_loss.py`
4. 對比結果，選擇配置
