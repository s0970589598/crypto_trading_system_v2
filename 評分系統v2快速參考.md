# 評分系統 v2.0 快速參考

## 評分公式

總分 = (進場質量 + 出場質量 + 風險管理 + 紀律遵守) / 4

每項初始分數：100 分

---

## 1️⃣ 進場質量（Entry Score）

### 扣分項
| 情況 | 扣分 |
|------|------|
| 逆勢交易（強趨勢） | -25 |
| 震盪市交易 | -10 |
| RSI 超買/超賣（非趨勢） | -20 |
| RSI 超買/超賣（趨勢中） | -5 |
| MACD 信號相反 | -15 |
| 高波動 + 高槓桿 | -15 |
| 布林帶極端位置（非突破） | -10 |

### 加分項
| 情況 | 加分 |
|------|------|
| 多重確認（3+） | +15 |
| 雙重確認 | +10 |
| 單一確認 | +5 |
| 強勢突破布林帶 | +5 |

---

## 2️⃣ 出場質量（Exit Score）

### 扣分項（根據交易風格）
| 情況 | 扣分 |
|------|------|
| 實際收益不足（嚴重） | -20 |
| 實際收益不足（一般） | -10 |
| 實際虧損過大 | -35 |
| 實際虧損較大 | -25 |
| 實際虧損超標 | -15 |
| 盈虧比不佳 | -15 |
| 虧損超過預期風險 | -20 |

### 加分項
| 情況 | 加分 |
|------|------|
| 優秀收益（≥ 20%） | +20 |
| 良好收益（≥ 10%） | +10 |
| 不錯收益（≥ 5%） | +5 |
| 優秀盈虧比 | +10 |
| 良好盈虧比 | +5 |
| 嚴格執行止損 | +10 |

---

## 3️⃣ 風險管理（Risk Score）

### 扣分項
| 情況 | 扣分 |
|------|------|
| 槓桿極高（> 100x） | -40 |
| 槓桿過高（> 50x） | -30 |
| 槓桿偏高（> 20x） | -20 |
| 手續費過高（> 50%） | -15 |
| 手續費偏高（> 30%） | -10 |

### 加分項
| 情況 | 加分 |
|------|------|
| 保守槓桿（≤ 5x） | +10 |
| 適中槓桿（≤ 10x） | +5 |
| 手續費控制良好（< 10%） | +5 |

---

## 4️⃣ 紀律遵守（Discipline Score）

### 扣分項
| 情況 | 扣分 |
|------|------|
| 爆倉 | = 0（其他分數也降低） |
| 虧損快速手動平倉（< 30分鐘） | -30 |
| 虧損較快手動平倉（< 2小時） | -15 |
| 虧損長時間手動平倉 | -5 |
| 獲利太少手動平倉 | -15 |
| 持倉時間極短（< 6分鐘） | -10 |
| 持倉時間過長（> 72小時） | -5 |

### 加分項
| 情況 | 加分 |
|------|------|
| 自動止損/止盈 | +15 |
| 持倉時間合理（符合風格） | +5 |

---

## 交易風格標準

| 風格 | 持倉時間 | 最小獲利 | 最大虧損 | 盈虧比 |
|------|---------|---------|---------|--------|
| 超短線 | < 1h | 0.2% × 槓桿 | 0.5% × 槓桿 | 1:1 |
| 日內 | 1-4h | 0.5% × 槓桿 | 1.0% × 槓桿 | 1.5:1 |
| 短線 | 4-24h | 1.0% × 槓桿 | 2.0% × 槓桿 | 2:1 |
| 中長線 | > 24h | 2.0% × 槓桿 | 3.0% × 槓桿 | 3:1 |

---

## 實際收益率計算

```
實際收益率 = 盈虧百分比 × 槓桿
```

**示例**：
- 盈虧：+0.5%
- 槓桿：10x
- 實際收益：0.5% × 10 = 5%

## 評分等級

| 總分 | 等級 | 說明 |
|------|------|------|
| 90-100 | 優秀 | 非常好的交易 |
| 80-89 | 良好 | 好的交易 |
| 70-79 | 及格 | 可接受的交易 |
| 60-69 | 需改進 | 有明顯問題 |
| < 60 | 不及格 | 嚴重問題 |
