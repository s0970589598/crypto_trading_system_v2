## ❓ 常見問題

### Q1: 為什麼要分成長週期和短週期？

A: 
- 長週期（15m, 1h, 4h, 1d）：用於回測和策略分析，數據量適中
- 短週期（1m, 3m, 5m）：用於進出場點分析，數據量大，按需獲取

### Q2: 為什麼不用 BingX API？

A: BingX API 無法批次獲取歷史數據，只能獲取最近 1-5 天的數據。Binance API 支持獲取任意歷史數據。

### Q3: Binance 和 BingX 數據差異大嗎？

A: 通常差異 < 0.1%，技術指標分析不受影響。

### Q4: 需要刪除舊的數據文件嗎？

A: 不需要，舊的數據文件（market_data_*.csv）可以保留，不影響新腳本。
