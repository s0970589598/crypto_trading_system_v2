### 1. 在 `MarketAnalyzer` 中添加新方法

#### `detect_missing_gaps(symbol, interval)` 
檢測數據中的缺失K線

**返回**：
```python
[
    {
        'start_time': datetime,    # 缺失開始時間
        'end_time': datetime,      # 缺失結束時間
        'missing_count': int       # 缺失數量
    },
    ...
]
```

**使用示例**：
```python
from src.analysis.market_analyzer import MarketAnalyzer

analyzer = MarketAnalyzer()
gaps = analyzer.detect_missing_gaps('BTCUSDT', '1h')

if gaps:
    total_missing = sum(gap['missing_count'] for gap in gaps)
    print(f"發現 {len(gaps)} 個缺失時間段，共 {total_missing} 個缺失K線")
```

#### `fill_missing_data(symbol, interval)`
檢測並自動填補缺失的K線數據

**返回**：
```python
{
    'total_missing': int,      # 總缺失數量
    'filled': int,             # 成功填補數量
    'gaps': List[Dict],        # 剩餘缺失的時間段
    'success': bool            # 是否完全修復
}
```

**使用示例**：
```python
analyzer = MarketAnalyzer()
result = analyzer.fill_missing_data('BTCUSDT', '1h')

if result['success']:
    print(f"✅ 成功填補 {result['filled']} 個缺失K線")
else:
    print(f"⚠️ 填補了 {result['filled']}/{result['total_missing']} 個K線")
```
