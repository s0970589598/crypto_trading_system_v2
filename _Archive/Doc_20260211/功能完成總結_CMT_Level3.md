## 🎯 核心功能

### 1. 大級別趨勢判斷
```
📊 1H/4H 大級別趨勢
趨勢判斷：強勢下跌
EMA 空頭排列，價格持續創新低
```

### 2. 開倉理由評估（含逆勢警告）
```
🎯 開倉理由評估
進場類型：逆勢摸底
評估：高風險
⚠️ 大級別下跌時做多，風險較高
```

### 3. K 線反轉信號識別
```
📈 15m/5m 入場級別信號
✅ 發現反轉信號：RSI 超賣 (<30)、MACD 金叉（看多信號）

或

❌ 無明確反轉信號，建議空倉等待
```

### 4. 市場結構分析
```
🏗️ 市場結構分析
結構類型：Lower Lows (空頭結構)
結構描述：價格持續創新低，反彈不破前高
```

### 5. 技術指標分析
```
📉 技術指標分析
布林帶位置：價格跌破下軌，可能超賣，💡 警惕反彈
動能分析：動能減弱（看空）
關鍵指標：RSI: 28.5, MACD: 死叉
```

### 6. 綜合評估
```
✅ 綜合評估
✅ 符合進場條件
或
⚠️ 逆勢交易，風險極高，建議空倉
或
❌ 無明確信號，建議空倉等待
```

### 7. 專業交易計劃
```
📋 專業交易計劃

進場區間：
- 當前價格：$69,102.90
- 建議進場：等待回調至 $68,500 (支撐位附近)

止損價格 (觸發後必須無條件執行)：
- 止損位：$68,200 (支撐位下方)
- 止損距離：902.90 (1.31%)

止盈價格：
- 第一止盈位（平倉 50%）：$70,500 (阻力位前)
- 第二止盈位（推保護止損）：建議在第一止盈後，將止損移至成本價

盈虧比 (R/R)：
- 盈虧比：1.55
- 評估：✅ 良好 (>1.5)

關鍵價位：
- 支撐位：$68,350 (距離 1.09%)
- 阻力位：$70,800 (距離 2.46%)

建議倉位：
- 基於 2% 風險管理原則
- 單筆最大虧損 = 總資金 × 2%
- 建議倉位 = (總資金 × 2%) ÷ 止損金額
- 例如：10,000 USDT 本金 → 最大虧損 200 USDT
  → 倉位 = 200 ÷ 902.90 = 0.2215 單位
```

## 📊 您的案例分析

根據您提供的數據：
```
當前價格：$69,102.90
多時區趨勢：
- 1d 🔴 strong_downtrend
- 4h 🔴 strong_downtrend
- 1h 🔴 strong_downtrend
- 15m 🔴 strong_downtrend
- 5m 🟢 uptrend
- 3m 🟢 uptrend
- 1m 🟢 strong_uptrend
```

**系統現在會分析為**：

1. **大級別趨勢**：強勢下跌（1d/4h/1h 都是 strong_downtrend）
2. **開倉理由**：
   - 如果建議做多 → 逆勢摸底（⚠️ 高風險）
   - 如果建議做空 → 順勢回調（✅ 有效）
3. **多空力量**：空頭優勢 -1（4個看空 vs 3個看多）
4. **綜合建議**：
   - 如果 RSI > 60 → 做空（順勢）
   - 如果 RSI < 60 → 觀望（等待信號）
   - 不會建議做多（因為是逆勢，會警告）
