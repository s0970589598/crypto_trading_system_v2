### 2. 減少交易頻率的配置

```bash
# 修改策略配置，減少交易頻率
nano strategies/multi-timeframe-aggressive.json

# 修改這些參數：
{
  "parameters": {
    "stop_loss_atr": 2.0,      // 從 1.5 改為 2.0（更寬止損）
    "take_profit_atr": 4.0,    // 從 3.0 改為 4.0（更高目標）
    "rsi_oversold": 25,        // 從 30 改為 25（更嚴格）
    "rsi_overbought": 75       // 從 70 改為 75（更嚴格）
  },
  "risk_management": {
    "position_size": 0.15,     // 從 0.20 改為 0.15（更小倉位）
    "leverage": 3,             // 從 5 改為 3（更低槓桿）
    "max_consecutive_losses": 2  // 從 3 改為 2（更早暫停）
  }
}

# 這樣配置會：
# - 減少交易頻率（更嚴格的進場條件）
# - 降低風險（更小倉位和槓桿）
# - 提高勝率（更寬的止損）
```

### 3. 設置每日虧損限制

```bash
# 修改系統配置

# 添加嚴格的風險控制：

# 這樣可以：
# - 防止單日大虧

```

## 🚨 重要提醒

### 必須做到的事

1. ✅ **嚴格執行止損**
   - 系統設置的止損不能移動
   - 觸及止損立即平倉
   - 不要心存僥倖

2. ✅ **不要手動干預**
   - 信任系統信號
   - 不要提前出場
   - 不要加倉

3. ✅ **記錄每筆交易**
   - 使用覆盤系統
   - 分析成功和失敗
   - 持續改進

4. ✅ **控制情緒**
   - 連損後不要報復性交易
   - 大賺後不要過度自信
   - 保持冷靜

5. ✅ **小資金開始**
   - 先用 10-20 USDT 測試
   - 確認有效後再增加
   - 不要一開始就大倉位
